Material Design Colors

Color in material design is inspired by bold hues juxtaposed with muted environments, deep shadows, and bright highlights. Material takes cues from contemporary architecture, road signs, pavement marking tape, and athletic courts. Color should be unexpected and vibrant.

Background Colors

Use the .bg- class prefix to paint any element with a Material Design color. You may also use the .bg-after and .bg-before prefix for pseudo elements. Examples:

.bg-red
.bg-green
.bg-blue
.bg-purple-700

Foreground Colors

Likewise Use the .fg- class prefix to color any text with a Material Design color. You may also use the .fg-after and .fg-before prefix for pseudo elements. Examples:

.fg-red
.fg-green
.fg-blue
.fg-purple

Border Colors

Use the .bdc- class prefix to paint the border of any element with a Material Design color. You may also use the .bdc-after and .bdc-before prefix for pseudo elements

.bdc-red
.bdc-green
.bdc-blue-600
A blockquote with .bdc-purple class applied

Button Colors

Use a combination of the .btn- class prefix with a Material Design color to create colored buttons. For example:

btn-orange
btn-cyan
btn-teal-600

Navigation Colors

Use the .uninav- navigation class prefix with a Material Design color to give your navigation a color theme

Gradients

Use either the .grad- class to access a number of premade gradients. Use .grad-tr- to switch the gradient direction to the right.

.grad-purple-indigo
.grad-tr-blue-cyan
.grad-tr-red-pink

To create your own gradient, simply append your values to the $md-grads variable and the relevant classes will be auto generated.


    $grads:(
      red-pink: ( 
        stops:(red, pink),
        fallback: mix(red, pink)
      ),
      purple-cyan: ( 
        stops:(purple, cyan),
        fallback: mix(purple, cyan)
      ),
      rainbow:(
        stops:(red, orange, yellow, green, blue, violet),
        fallback: mix(red, orange, yellow, green, blue, violet)
      )
    );

Disabling Specific Colors

The Material Design color classes are generated by a Sass mixin that reads from map variable called $md-colors. To disable a certain color, simply locate the color in the map and set the active:true entry to active:false.


  $md-colors: (
    "red": (
      active: false,
      500: #f44336,
     ),
     "pink": (
      active: true,
      500: #e91e63,
     )
   );
